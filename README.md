# Excel Tools (Office Add-in)

Excel用のOffice Add-inです。React + TypeScript + Viteで構築されています。

## 起動・実行・デバッグ方法

### 開発サーバーの起動

```bash
npm run dev
```

### デバッグ実行

開発サーバーを起動した後、**別のターミナルタブ**で以下を実行：

```bash
npm run debug:desktop
```

これにより、Excel Desktopアプリケーションでアドインが起動し、デバッグが可能になります。

### デバッグの停止

```bash
npm run stop
```
